Thanks for installing this package. 

The package will create a new hidden data type property which will store the UUID. 

Here is the quick steps to start using this document type for the starter 

Steps:
1. Go to Developer Section in Umbraco Backoffice and create a new custom Data Type
2. Click the 'Property editor' and select the 'Sanz.UUID Generator' data type
3. Save it with some custom name. 
4. Go to 'DocumentType' and add new custom field reusing the newly created custom data type. 
5. You will now have a new UUID field, which will auto generate the unique uid.
